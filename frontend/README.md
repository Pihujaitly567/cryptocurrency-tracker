# CripTik Frontend

React-based cryptocurrency tracker frontend application.

## Features

- Real-time cryptocurrency prices
- Interactive charts
- Portfolio management
- Favorites system
- Currency converter
- Market news

## Tech Stack

- React 19
- Vite 7
- TailwindCSS 4
- Recharts
- Axios

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Opens on http://localhost:5173

## Build

```bash
npm run build
```

## Environment

Create `.env`:
```
VITE_API_URL=http://localhost:5000/api
```
